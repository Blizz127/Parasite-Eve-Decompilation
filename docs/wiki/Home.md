# Parasite Eve (PS1) Decompilation

USA NTSC-U decompilation research for **Parasite Eve** (SLUS-00662 / SLUS-00668).
This is a matching decomp effort — not a PC port.

## What this project is

- One canonical boot executable (`SLUS_006.62` on disc 1; disc 2's EXE is byte-identical).
- Splat-based disassembly, asm-only rebuild harness, and incremental C replacement.
- Reproducible scripts and configs; **no game data in git**.
